    Visual Example
    Font Selection
    Tips

    How to Download Fonts from DaFont.com:

    1. Visit dafont.com

    2. Search for fonts that match your theme

    3. Click "Download" button next to your chosen font

    4. Extract the ZIP file to your computer

    5. For Windows: Right-click the font file → Install

    6. For Mac: Double-click the font file → "Install Font"

    7. The font will now appear in your design software

    Note: Do not use any Chalkboard based font

    Visual Consistency Tips:

    • Choose 2-3 fonts maximum (one for headings, one for body text)

    • Select a consistent color palette that relates to your theme

    • Use the same layout grid structure throughout

    • Maintain consistent spacing between elements

    • Use similar photo treatments or filters across images

    • Keep page numbering and header/footer styles consistent

  • Source References: Have you properly referenced all images and information using Harvard referencing?
    Visual Example
    Referencing Guide

    Referencing Tips:

    • Include direct references under each image

    • Image format: Designer/Photographer (Year) Title

    • Website format: Author (Year) 'Title of page', Website name. Available at: URL (Accessed: date)

    • Book format: Author (Year) Title. Publisher, Place of publication, p.##

    • Harvard Referencing Guide: https://hsstokes.github.io/Harvard-Referencing-Guide/

    • Include a full bibliography as the last page of your portfolio
